# Design and Copywriting

- Maintain sufficient contrast and prefer perceptual contrast checks over simplistic numerical thresholds when available.
- Keep charts and status colors usable for color-blind users.
- Match browser chrome to the page background when theme color metadata matters.
- Set the appropriate `color-scheme` so form controls and scrollbars match the intended theme.
- Use active voice and action-oriented labels.
- Prefer clear, specific button text over vague continuation labels.
- Keep nouns and terminology consistent across a flow.
- Use numerals for counts and keep number-unit formatting consistent.
- Use positive, solution-oriented language where possible, especially in failure states.
- Error messages should explain what went wrong and what the user can do next.
- Favor concise copy. Remove words that do not help the user act or understand.
