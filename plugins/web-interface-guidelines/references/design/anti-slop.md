# Anti-Slop

- Reject interchangeable SaaS layouts that could belong to any product.
- Avoid generic font stacks unless the host product intentionally chooses them.
- Avoid predictable component patterns when stronger composition is justified by the content and audience.
- Do not reuse the same palette, motion language, and aesthetic across unrelated interfaces.
- Distinctive minimalism is valid when it is intentional and coherent.
- Generic minimalism is not a substitute for a point of view.
- If the interface is meant to stand out, let typography, spacing, layout rhythm, and surface treatment do real work.
