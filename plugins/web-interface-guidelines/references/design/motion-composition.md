# Motion and Composition

- Use motion to reinforce hierarchy, cause and effect, and spatial continuity.
- Prefer a few meaningful animated moments over scattered micro-interactions that do not add understanding.
- Build personality through spacing, overlap, asymmetry, rhythm, and background treatment.
- Match visual complexity to the chosen direction; a restrained system should stay calm, while a richer direction can support more layering.
- Use gradients, texture, pattern, shadow, and depth when they strengthen the atmosphere instead of muddying it.
- Keep composition and motion aligned so the interface feels designed by one hand.
- Avoid decorative movement that competes with the content or makes the system feel busy.
