# Agent Orchestrator (Skill)

This folder contains the **agent-orchestrator** skill for Codex. It is designed for complex, multi-step engineering work where you want strict phase gates (audit → design → implement → review → validate → deliver).

## Expected agent roles

You should have these roles available:

- `architect`
- `auditor`
- `explorer` / `scout`
- `worker`
- `implementer`
- `spec_reviewer`
- `quality_reviewer`
